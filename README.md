# Emotion Memory 手機介面版

## 本機預覽
```bash
npm install
npm run dev
```

## 打包
```bash
npm run build
```

## Vercel 部署
1. 將全部檔案上載到 GitHub repository
2. Vercel 選擇該 repository
3. Framework Preset: Vite
4. Build Command: `npm run build`
5. Output Directory: `dist`

## GitHub Pages
```bash
npm run build
```
然後部署 `dist` 目錄。
